# Airbnb API

REST & GraphQL API of the Airbnb Clone using Django REST Framework and Graphene GraphQL

### API Actions

- [x] List Rooms
- [x] Create Room
- [x] See Room
- [x] Edit Room
- [x] Delete Room
- [x] See Profile
- [x] Edit Profile
- [x] Add / Remove Room to Favourites
- [x] Create Account
- [x] Login(JWT)
- [x] JWT Authentication
- [x] Search Rooms
